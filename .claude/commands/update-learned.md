# Update Learned Context

Review our conversation history from this session and append any of the following to `learned.md` if they are not already documented:

- Architecture or design decisions made
- Bugs solved and how

- Be concise — bullet points preferred over paragraphs
- Skip anything trivial or already covered
- Add a date stamp to any Session Notes entry: `### YYYY-MM-DD`
- Do not update `CLAUDE.md`
- Confirm when done and list what was added
